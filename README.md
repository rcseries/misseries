# Series Tracker

Aplicación web para llevar control de las series que ves.

## Características

- 📺 Categorización de series (Pendiente de estreno, En emisión, A medias, Pendientes, Vistas)
- 📅 Checklist automático de capítulos para series en emisión
- ⭐ Sistema de calificación para series vistas
- 🎨 Diseño responsive con tema oscuro
- 🔥 Base de datos en tiempo real con Firebase

## Configuración

1. Crear proyecto en Firebase Console
2. Configurar Firestore Database
3. Copiar credenciales en `js/config.js`

## Uso

Abrir `index.html` en un servidor web.
